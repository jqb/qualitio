The main goal of Qualitio project is to unleash MeeGo QA community. Qualitio is an open-source tool which allows testing community to contribute into Meego project with an ease and pleasure.

How to start?

Just login with your Google or Yahoo account or register a new Qualitio account.

What's next?

Then please check Qualitio wiki page and... that's it! You can start testing! Prepare test cases for requirements that you want to test, execute them and compose complex report.

Your contribution makes a difference! You can start testing today and have an influence on the MeeGo project. Qualitio is an open-source web application. More information can be found on Qualitio web page.

If you want to know more about MeeGo, please check the official MeeGo website. MeeGo™ is a trademark of the Linux Foundation.
